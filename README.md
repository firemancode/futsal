# 🏟️ Futsal Center - Website Penyewaan Lapangan Futsal

Website modern dan responsif untuk menampilkan informasi penyewaan lapangan futsal dengan fitur booking online.

## ✨ Fitur Utama

- **Design Modern & Responsif** - Tampilan yang menarik dan berfungsi sempurna di semua perangkat
- **Informasi Lapangan** - Detail lengkap setiap lapangan dengan status ketersediaan
- **Daftar Harga** - Harga transparan berdasarkan waktu (pagi, siang, malam)
- **Sistem Booking Online** - Form booking yang user-friendly dengan validasi
- **Kalkulasi Harga Real-time** - Estimasi harga otomatis berdasarkan pilihan
- **Animasi & Interaksi** - Efek visual yang smooth dan menarik
- **Mobile-First Design** - Optimized untuk penggunaan mobile

## 🚀 Teknologi yang Digunakan

- **HTML5** - Struktur semantik yang bersih
- **CSS3** - Styling modern dengan Flexbox dan Grid
- **JavaScript (ES6+)** - Interaktivitas dan validasi form
- **Font Awesome** - Icon library
- **Google Fonts** - Typography yang menarik
