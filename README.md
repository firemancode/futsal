# Futsal Center - Website Penyewaan Lapangan Futsal

Website modern dan responsif untuk menampilkan informasi penyewaan lapangan futsal dengan fitur booking online.

## 🚀 Fitur Utama

### 📱 Responsive Design
- Fully responsive untuk desktop, tablet, dan mobile
- Mobile-first approach dengan hamburger menu
- Smooth scrolling navigation

### 🏟️ Informasi Lapangan
- Tampilan 3 jenis lapangan (Indoor A, Indoor B, Outdoor)
- Status ketersediaan real-time
- Fitur-fitur lapangan dengan ikon
- Gambar lapangan yang menarik

### 💰 Daftar Harga
- Harga weekday dan weekend
- Paket promo (2 jam, 4 jam, seharian)
- Tampilan harga yang jelas dan terstruktur

### 📅 Sistem Booking
- Form booking yang user-friendly
- Validasi form yang lengkap
- Modal konfirmasi booking
- Kalkulasi harga otomatis

### 📞 Informasi Kontak
- Detail kontak lengkap
- Peta lokasi (Google Maps)
- Jam operasional
- Social media links

## 🛠️ Teknologi yang Digunakan

- **HTML5** - Struktur website
- **CSS3** - Styling dan animasi
- **JavaScript (ES6+)** - Interaktivitas dan validasi
- **Font Awesome** - Ikon-ikon
- **Google Fonts** - Typography

## 🔧 Cara Menjalankan

1. Clone atau download repository ini
2. Buka file `index.html` di browser
3. Website siap digunakan!

```bash
# Atau jika menggunakan live server
npx live-server
```
